# Design System & Documentation

## Imported HTML Links
These are the external resources linked in the `index.html` file.
``` html
    <link href="https://fonts.cdnfonts.com/css/gilroy-bold" rel="stylesheet">
    <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css">
```

## Root Variables (CSS)
These variables are defined in the `:root` selector in `assets/style.css` and are used throughout the site for consistency.

### Colors
``` css
    --primary-color: #E58411;
    --text-dark: #1E1E1E;
    --text-gray: #8D8D8D; 
    --bg-light: #F7F7F7;
    --white: #FFFFFF;
```

### Fonts
``` css
    --font-heading: 'Gilroy-Bold', sans-serif;
    --font-body: 'Inter', sans-serif; 
```

### Spacing & Layout
``` css
    --container-width: 80rem;
    --section-spacing: 6.25rem; 
```

### Transition
``` css
    --transition: all 0.3s ease;
```
